# 🌈 童童岛奇遇记 · 部署指南

## 📦 这个文件夹里都有什么

```
dist/
├── index.html        ← 主入口（HTML + JS + CSS 全在里面）
├── manifest.json     ← PWA 配置（让浏览器知道这是 App）
├── sw.js             ← Service Worker（实现离线缓存）
├── icon.svg          ← 矢量图标
├── icon-192.png      ← 192×192 图标（Android 主屏幕用）
├── icon-512.png      ← 512×512 图标（启动屏 / iOS 主屏幕用）
└── README.md         ← 这份指南
```

**所有文件加起来约 200KB**，部署到任何静态托管平台都行。

## 🚀 三种推荐部署方式

下面三种从简到难，按你的需求选一种就行。

### 方案 A：GitHub Pages（最简单 · 免费）

适合：给亲朋测试、海外用户、你只想验证一下产品

**步骤**：

1. **注册 GitHub 账号**（https://github.com）
2. **创建新仓库**：
   - 点右上角 + → New repository
   - Repository name：`kidquest`（或任何你喜欢的名字）
   - 选 Public（GitHub Pages 免费版要求公开仓库）
   - 创建
3. **上传 dist 文件夹里的所有文件**：
   - 仓库页面点 "Add file" → "Upload files"
   - 把 dist 里的所有文件（不是 dist 文件夹本身）拖进去
   - 滚到底点 "Commit changes"
4. **开启 GitHub Pages**：
   - 仓库点 Settings → Pages
   - Source 选 "Deploy from a branch"
   - Branch 选 `main`（或 `master`），Folder 选 `/ (root)`
   - 点 Save
   - 等 1-2 分钟，会得到一个网址，类似：
     `https://你的用户名.github.io/kidquest/`
5. **打开网址试试**！手机访问后，浏览器右上角菜单里会有"添加到主屏幕"

**注意**：
- 国内访问可能慢（GitHub 在海外）
- 自带 HTTPS，PWA 功能完整可用
- 修改后重新上传即可，几分钟自动更新

---

### 方案 B：Vercel（更快 · 免费 · 支持自定义域名）

适合：海外用户、需要更快的访问速度

**步骤**：

1. 注册 Vercel（https://vercel.com）—— 用 GitHub 账号登录最方便
2. 点 "Add New" → "Project"
3. 选你的 kidquest 仓库 → Import
4. 一路 Next（默认配置就行）
5. 几十秒后会得到一个网址，类似：
   `https://kidquest-xxx.vercel.app`
6. 也可以在 Settings → Domains 里绑定你自己的域名

**优势**：比 GitHub Pages 快、CDN 全球加速、自动 HTTPS。

---

### 方案 C：腾讯云 / 阿里云 静态托管（国内最快）

适合：主要给国内用户用、想用 .cn 域名

**腾讯云 COS 步骤简述**：

1. 注册腾讯云账号 + 实名认证
2. 开通 **对象存储 COS**（按量付费，1GB 月费几毛钱）
3. 创建一个**公有读私有写**的存储桶，地域选"国内"
4. 进入存储桶 → 文件列表 → 上传 dist 里所有文件
5. 进入存储桶 → 基础配置 → 开启"静态网站"
6. 把"索引文档"设为 `index.html`
7. 拿到一个访问 URL（类似 https://xxx.cos-website.ap-shanghai.myqcloud.com）

**关键提醒**：
- 国内服务器要求**域名备案**才能绑定自定义域名（个人备案约 7-15 天）
- 腾讯云 / 阿里云的 COS 上传也支持命令行（更高效），你可以问 ChatGPT 怎么用

## 🔍 部署后怎么测试

部署完后用手机打开网址，**重点测这几件事**：

### ✅ 基础测试
- [ ] 启动屏正常显示 2-3 秒
- [ ] 选伙伴页面正常
- [ ] 各岛屿能进入
- [ ] 答题、闯关、解锁图鉴正常

### ✅ PWA 测试
- [ ] 浏览器右上角菜单里有"添加到主屏幕"选项（iOS Safari）
- [ ] 添加后桌面有"童童岛"图标
- [ ] 点桌面图标，App 全屏运行（无浏览器地址栏）
- [ ] **关掉网络/打开飞行模式后，App 仍然能打开和玩**（这是 PWA 离线的关键测试）

### ✅ 多设备测试
- [ ] iPhone Safari
- [ ] Android Chrome
- [ ] iPad Safari
- [ ] 电脑浏览器

## ⚠️ 部署前的法律注意事项

如果只给亲朋测试，下面可以跳过。如果**正式发布给陌生用户**：

### 需要做的：
1. **隐私政策**：哪怕你不收集任何数据（这个 App 也确实不收集），也建议明示这一点
2. **儿童信息保护**：中国《个人信息保护法》对儿童信息（14 岁以下）有更严格规定。你这个 App 不收集任何信息，完全本地存储，相对安全
3. **域名备案**：用国内服务器 + 国内域名（.cn / .com.cn）必须备案；用境外服务器则不强制

### 建议先不做的：
- 不要急着加广告（影响家长口碑）
- 不要急着做付费（先验证产品价值）
- 不要急着收集用户数据（合规复杂）

```
  ↓ 准备打包成原生 App
```
